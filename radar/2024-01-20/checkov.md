# Checkov

<a href="https://github.com/nimrodkor/checkov" target="_blank">Checkov</a> is a static code analysis tool for 
infrastructure-as-code. It scans cloud infrastructure provisioned using Terraform, Terraform plan, Cloudformation, 
Kubernetes, Dockerfile, Serverless or ARM Templates and detects security and compliance misconfigurations.
